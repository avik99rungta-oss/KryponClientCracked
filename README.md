DonutSMP Hacks
A simple Fabric mod that opens an in-game menu using the Right Shift key.

Requirements

Minecraft (Java Edition)

Fabric Loader installed

Fabric API installed

Setup Instructions

Install Fabric Loader for your Minecraft version

Download and place Fabric API into your mods folder

Add this mod’s .jar file into the same mods folder

Open the Minecraft Launcher

Select your Fabric profile

Launch the game

How to Use

Join DonutSMP

Press Right Shift to open the menu

Press Right Shift again to close it


Troubleshooting
Menu not opening

Make sure you launched the Fabric profile

Check that Fabric API is installed

Verify no keybind conflicts with Right Shift

Game crashing

Ensure mod version matches your Minecraft version

Update Fabric Loader and Fabric API if needed

Wrong key opens menu

Rebind the key in Options → Controls → Key Binds
